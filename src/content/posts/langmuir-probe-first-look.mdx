When I first read a Langmuir probe trace, the most useful mental picture was simple: the curve is a compressed map of which particles can reach the probe at a given bias.

In the electron-retarding region, a common first approximation is

$$
I_e(V) \propto \exp\left(\frac{e(V - V_p)}{k_B T_e}\right).
$$

That means the semi-log slope gives a direct handle on the electron temperature.

```python
import numpy as np

def estimate_te(voltage, current):
    coeffs = np.polyfit(voltage, np.log(current), 1)
    return 1 / coeffs[0]
```
